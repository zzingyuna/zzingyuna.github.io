# Java

#### 환경변수 cmd에서 바로 세팅
```
재부팅 X
c:\> set classpath=%classpath%;.
c:\> set JAVA_HOME=c:\myjava\jdk1.5
c:\> set path=%path%;%JAVA_HOME%bin
c:\> javac Test.java
```
