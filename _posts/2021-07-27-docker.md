### attach
/bin/bash를 실행한것과 같음, 명령을 자유롭게 입력 가능.  
DB나 서버 애플리케이션을 실행하면 입력 불가 출력만 볼 수 있음.  
```
sudo docker attach [컨테이너명]
```  


### 컨테이너 빠져나오기

```
exit #컨테이너 종료됨
ctrl+D #컨테이너 종료됨

ctrl+P, ctrl+Q
```
